# 更新日志

所有重要变更都会记录在此文件中。


## [1.0.0] - 2026-02-12

### 新增

- 初始版本发布
- 支持通过行政区划代码获取天气数据（省/市/区县三级级联选择）
- 支持通过 Home Assistant 地点经纬度获取天气数据
- 实时天气实体：温度、体感温度、湿度、气压、风速、风向、云量、能见度等
- 7天逐日天气预报（`weather.get_forecasts` daily 服务）
- 24小时逐小时天气预报（`weather.get_forecasts` hourly 服务）
- 气象预警独立传感器实体
- 空气质量指数（AQI）独立传感器实体
- PM2.5、PM10、NO₂、SO₂、O₃、CO 空气质量子指标
- 配置流支持 AK 密钥连通性校验与重复配置拦截
- 选项流支持更新间隔配置（300-7200秒）
- 中文（zh-Hans）与英文（en）国际化支持
- 诊断导出支持（自动脱敏 AK）
